# 소개
안녕하세요. 디스코드 오분가 봇 입니다. 제작 언어는 Python3.11이며, DB는 NoSQL인 Google Firebase.Firestore를 사용하였습니다.

<br>

## 기능
롤 / 롤체 전적검색, 낚시, 외화거래, 코인거래, 은행거래, 레벨업, 유저  PVP 등 디스코드 서버를 더 재미있게 하고,디스코드 채널의 유저들이 더 적극적으로 서버 내에서 활동할 수 있기를 바라며 만들어 봤습니다.
